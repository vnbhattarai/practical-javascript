# Functions in JS are just like recipes

### Make turkey Sandwich

- Get one slice of bread
- Add turkey
- Put a slice of bread on top

### Code example:

```javascript
function makeTurkeySandwich() {
    Get one slice of bread;
    Add turkey;
    Put a slice of bread on top;
}
makeTurkeySandwich();   // Run the steps in function makeTurkeySandwich
```
